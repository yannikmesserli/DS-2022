From editing in 2D to editing in 3D

  • Using the Editor widget in a SceneView works out of the box.
  • Using the SketchViewModel also works 3D.
  • FeatureForm and Editor widget is the same in 2D and 3D.
  • No need to prepare data specifically for 3D.
                
                  const view = new SceneView({ /* ... */ });
                  // const view = new MapView({ /* ... */ });

                  const editor = new Editor({ view /* ... */})

Elevation Mode - on the ground

                
                const trees = new GraphicsLayer({ 
                  elevationInfo: { mode: "on-the-ground" }
                });
                view.map.add(trees);

                // Get a `PointSymbol3D` for our graphics.
                const treeSymbol = await new WebStyleSymbol({
                  name: "Larix",
                  styleName: "EsriRealisticTreesStyle",
                }).fetchSymbol();

                const button = document.createElement("calcite-button");
                button.textContent = "Add tree";

                button.addEventListener("click", () => {
                  sketchViewModel.layer = trees;
                  sketchViewModel.pointSymbol = treeSymbol;
                  sketchViewModel.create("point");
                });
